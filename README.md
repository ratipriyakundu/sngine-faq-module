# Sngine FAQ Module

Multilingual FAQ Module for Sngine Social Network Platform

## Installation

```bash
composer require ratipriya/sngine-faq-module
```

## Usage

```php
<?php

require_once __DIR__.'/vendor/autoload.php';

use Ratipriya\SngineFAQ\SngineFAQ;

$faq = new SngineFAQ();

```
